# sample_express_app

ExpressとTypeScriptで構成されたサンプルアプリケーションです。

## 必要環境

- Node.js : v20.16.0
- npm : 10.8.1

## セットアップ

```bash
npm install
npm run serve
```

## スクリプト一覧
- npm run build : TypeScriptのビルド
- npm run start : ビルドして dist/index.js を起動
- npm run serve : ビルドして dist/server.js を起動
- npm run test : Jestによるテスト
